# Agent Skills

A collection of reusable skills for AI coding agents.

## Available Skills

- **generating-claude-instructions** — Generate a CLAUDE.md file at the project root by deeply exploring the actual source code.

## Skill Discovery

Skills are located in the `skills/` directory. Each skill has its own subdirectory containing a `SKILL.md` file that describes when and how to use it.

When a user asks you to perform a task that matches a skill's "When to Use" criteria, follow the process described in the corresponding `SKILL.md`.
